# Python Game Assessment – CSC-44102

## Overview
This repository contains a Python-based number guessing game developed for the module **Foundations of Programming and Software Engineering (CSC-44102)**.

The project demonstrates basic programming concepts, clean coding practices, and the use of Git and GitHub for version control.

---

## Game Description

The project includes two versions of a number guessing game:

### 1. Terminal Version (`game.py`)
- User guesses a number between 1 and 100
- Input validation for correct entries
- Limited number of attempts (10)
- Tracks:
  - Total games played
  - Total games won
- Option to replay the game

---

### 2. GUI Version (`gui_game.py`)
- Built using Python's `tkinter` library
- Interactive graphical interface
- Features:
  - Input field for guesses
  - Feedback messages (too high / too low)
  - Attempts counter
  - Reset / New Game button

---

## Technologies Used

- Python
- Tkinter (for GUI)
- Git (version control)
- GitHub (repository hosting)
